damei.misc.list2table¶

damei.misc.list2table(a, float_bit=2, alignment='<')¶

a – list, 长度为n，每个元素内由是一个str bool int float None list或dict

Parameters

  • float_bit – 浮点数的小数位数

  • alignment – 表格对齐方式

Returns

format str:

Example
>>> COLUME1 COLUME2 COLUME3
>>> 1       True    3.14
