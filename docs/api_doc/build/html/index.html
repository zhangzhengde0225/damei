Welcome to damei’s documentation!¶

Damei is an Artificial Intelligence and Quantum Mechanics Intersection Python Library.

The open source codes are available on github.com/zhangzhengde0225/damei and offers a simple and intuitive API. Check out the Usage section for further information.
