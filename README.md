# damei

damei(大妹) library。

涉及深度学习和量子力学交叉。

提供常用函数、控制台、解算器等。

# 1.安装

pypi安装:

```
pip install damei  
```

[//]: # (或从源码安装：[Download]&#40;https://github.com/zhangzhengde0225/damei/archive/refs/heads/master.zip&#41;.)
下载源码后：

```
unzip damei-master.zip
cd damei-master
python setup.py install
```

## [详细API文档](http://47.114.37.111)

import damei as dm
